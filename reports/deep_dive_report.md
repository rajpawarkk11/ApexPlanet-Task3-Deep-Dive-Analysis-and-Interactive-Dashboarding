# Deep-Dive Analysis Report

## Objective

The objective of this project was to perform a deep-dive business analysis and build an interactive dashboard using Power BI.

## Core KPIs

1. Total Revenue
2. Total Orders
3. Total Customers
4. Average Order Value

## Customer Segmentation Analysis

Customer segmentation was performed using K-Means Clustering.

Three customer segments were identified:

### High Value Customers

- Highest average sales
- Highest quantity purchased
- Most valuable customer group

### Medium Value Customers

- Moderate sales contribution
- Moderate purchasing activity

### Low Value Customers

- Lowest revenue contribution
- Lowest purchasing activity

## Key Findings

- High Value customers generated the highest revenue.
- Electronics was the top-performing category.
- Patna generated the highest sales among cities.
- Male and Female customers contributed almost equally to revenue.
- Customer segmentation helped identify valuable customer groups for targeted business strategies.

## Dashboard Features

The Power BI dashboard includes:

- Core KPI Monitoring
- Revenue Analysis
- City-wise Sales Analysis
- Gender-wise Sales Analysis
- Customer Segmentation Insights

## Conclusion

The interactive dashboard enables stakeholders to monitor business performance and understand customer segments effectively.
